## 添加条目

条目新建你当然可以手动一个个加：[手动新增条目](./subjects.md#新增条目页)

但是更推荐安装 [番组计划插件](../plugins/plugin-bgmtv) 进行快捷添加条目：[番组计划插件#快速新增条目](../plugins/plugin-bgmtv#快速新增条目)

您可以选择批量绑定，或者手动绑定，更推荐批量绑定，

批量绑定（推荐）
