## 管理主面板

![Ikaros Console Plugins](/img/user-guide-plugins/Snipaste_2023-07-30_14-33-46.png)

- 重载所有：重启所有插件，不建议使用；
- 安装插件：打开上方安装插件抽屉，上传插件文件(一般是`.jar`包)，安装插件；
- 插件详情：前往插件详情页；
- 插件启动：启动插件；
- 插件停止：停止插件；
- 插件禁用：禁用插件，不建议使用，没啥意义，禁用后你点击启动一样能启动；
- 插件重载：重启插件；
- 插件卸载：删除插件；
- 插件重置：未实现功能；
